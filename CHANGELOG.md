# Changelog

All notable changes to PageMacroCenter are documented in this file.

## [0.3.4] - 2026-07-18

### Added

- Multi-word search across macro names and relative paths
- Search normalization for spaces, underscores and German umlaut spellings
- Version number in the PageMacroCenter window title

### Included

- Recursive page macro browser
- Native EPLAN macro preview
- Drag-and-drop insertion into EPLAN
- Refresh with `F5` and search reset with `Esc`
